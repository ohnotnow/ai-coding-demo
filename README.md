# AI Coding Demo

A satirical demo of "What Management Think AI Coding Is Like" - perfect for IT team presentations.

See [a demo](https://ai-coding-demo.netlify.app/) on Netlify.

## The Experience

1. Dramatic terminal
2. Spinning robot heads
3. Psychedelic Tempest-style background and "AI" writes grandiose code
4. Power messages flash
5. A dramatic ... climax?

## Quick Start

```bash
bun install
bun run index.ts
```

Open http://localhost:3000

## Requirements

- [Bun](https://bun.sh)
