# AI Coding Demo

A satirical demo of "What Management Think AI Coding Is Like" - perfect for IT team presentations.

## The Experience

1. Dramatic terminal types `MAKE ME AN APP!!!`
2. Spinning robot head "processes" with buzzword status messages
3. Psychedelic Tempest-style background while "AI" generates grandiose code
4. Power messages flash: "THE POWER OF AI", "SYNERGY ACTIVATED"
5. Ominous WarGames-style button: "DO YOU WANT TO PLAY A GAME????"
6. Click reveals... a `SyntaxError` on line 42

## Quick Start

```bash
bun install
bun run index.ts
```

Open http://localhost:3000

## Requirements

- [Bun](https://bun.sh)
